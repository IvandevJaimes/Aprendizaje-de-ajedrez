# Juego educativo de ajedrez (Concurso de preguntas) — Proyecto final Programación I

**Descripción breve**  
Juego educativo de consola que plantea preguntas de opción múltiple relacionadas con ajedrez. Diseñado como proyecto final para la materia Programación I. El objetivo es enseñar conceptos de ajedrez (movimientos, reglas, tácticas básicas) mediante un formato interactivo en consola.

**Stack / Tecnologías**
- Python (versión recomendada: 3.8+)
- Ejecución en terminal / consola

**Características**
- Interfaz por consola, texto claro y menú navegable.
- Banco de preguntas con opciones múltiples (A, B, C, D).
- Sistema de puntaje y feedback inmediato por respuesta.
- Niveles de dificultad (fácil, intermedio, avanzado) o selección aleatoria.
- Registro de puntajes por sesión (mostrar resultado final).
- Validación de entradas del usuario (evita crashes por entradas inválidas).
